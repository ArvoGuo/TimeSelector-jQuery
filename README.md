# TimeSelector-jQuery

A component of TimeSelector
Base On Jquery

##Demo

[Demo](http://c.oooooo6.com/TimeSelector-jQuery/index.html)

##Use

HTML

```html
<div class="time-selector test">
  <input type="text" class="time-selector-field">
</div>
```

JS

```javascript
var a = new $.TimeSelector({
  div: $('.time-selector').eq(0),
});
```
